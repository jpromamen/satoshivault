How SatoshiVault works

A trustless dead drop system built on OPNet Bitcoin L1 smart contracts.

01
🔐
Encrypt your secret
Write your secret message. We hash it client-side using SHA-256. Only the hash goes on-chain — the message stays with you until reveal.
02
⛓️
Set unlock conditions
Choose a block height (time-based) or a Bitcoin trigger address (payment-based). When conditions are met, the drop becomes revealable by anyone.
03
📡
Telegram alerts fire
Watchers subscribe via Telegram. The moment unlock conditions are met on Bitcoin L1, an automatic alert is sent to the OPMarket Alerts channel.
04
🌐
Anyone can reveal
Once unlocked, anyone can call reveal(). The original creator submits the plaintext message, which is verified against the on-chain hash.
05
₿
Settled on Bitcoin
All logic runs as an OPNet WASM smart contract on Bitcoin L1. No Ethereum. No bridges. No custodians. Just Bitcoin blocks.
06
🛡️
Censorship resistant
Once a dead drop is created, no one — not even SatoshiVault — can stop it from revealing. Bitcoin's immutability is the guarantee.
