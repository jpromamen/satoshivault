HOW SATOSHIVAULT WORKS

A trustless dead drop protocol on OPNet Bitcoin L1 WASM smart contracts.

01
🔐
ENCRYPT YOUR SECRET
Write your message. We SHA-256 hash it client-side. Only the hash goes on-chain. Your secret stays private until the vault opens.
02
⛏
BLOCK HEIGHT LOCK
Set a Bitcoin block height. When Bitcoin L1 mines past that block, the drop automatically unlocks — trustless time-release.
03
₿
PAYMENT THRESHOLD
Set a BTC amount. Payments to the trigger address fill a progress bar. Once the threshold is reached, the vault opens.
04
⚡
COMBINE CONDITIONS
Require BOTH a block height AND a payment threshold. Maximum security — both conditions must be satisfied simultaneously.
05
📡
TELEGRAM ALERTS
Watchers subscribe via @opmarket_alert. The instant the vault unlocks on Bitcoin L1, an automatic alert fires to all subscribers.
